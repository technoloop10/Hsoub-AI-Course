# ملخص مكتبة re في بايثون

## 1. التعامل مع السلاسل النصية الخام (Raw Strings)
* تظهر مشكلة "الشرطة المائلة العكسية" في بايثون.
* بايثون يفسر `\` كأمر نظام (مثل `\n` للسطر الجديد).
* لغة RegEx تستخدم `\` لتهريب الرموز (مثل `\d` للرقم).
* **الحل** هو استخدام النص الخام بوضع حرف `r` قبل النص.
* مثال: `r"\d{3}"` يخبر بايثون بعدم تفسير الشرطة المائلة.
* هذا يجعل الكود أنظف وأسهل في القراءة والمراجعة.

## 2. أهم الدوال في مكتبة re

| الدالة | الوظيفة |
|--------|----------|
| `re.search(pattern, text)` | تبحث عن أول تطابق في أي مكان داخل النص. |
| `re.match(pattern, text)` | تبحث عن التطابق في "بداية" السلسلة النصية فقط. |
| `re.findall(pattern, text)` | تستخرج كل التطابقات وتضعها في قائمة (List). |
| `re.finditer(pattern, text)` | تعيد كائن (Iterator) يحتوي على تفاصيل كل تطابق. |
| `re.sub(pattern, replacement, text)` | تستخدم لاستبدال النصوص بناءً على نمط معين. |
| `re.split(pattern, text)` | تقسم النص إلى أجزاء بناءً على وجود النمط. |

## 3. خصائص كائن التطابق (Match Object)
عند نجاح البحث، يتم إنشاء كائن يحتوي على البيانات التالية:
* `match.group()` — تعيد النص الكامل الذي تمت مطابقته بنجاح.
* `match.group(1)` — تعيد النص الموجود داخل القوس الأول (المجموعة الأولى).
* `match.start()` — تعيد رقم الفهرس (Index) الذي بدأ عنده التطابق.
* `match.end()` — تعيد رقم الفهرس الذي انتهى عنده التطابق.
* `match.span()` — تعيد البداية والنهاية معاً في شكل (Start, End).

## 4. مثال برمجي (للنسخ)

```python
import re

# نص خام لمنع مشاكل التهريب
pattern = r"^(\w+)@(\w+)\.([a-z]{2,4})$"
text = "user@gmail.com"

# محاولة المطابقة
result = re.search(pattern, text)

if result:
    print(result.group())
    print(result.start())
    print(result.end())
    print(result.group(1))
```

## مثال على نص مع findall

```python
import re

data_source = """
سجل الزوار اليوم:
1. المطور سامي: sami.dev@tech.com وهاتفه 00966 5544 12345678
2. المهندسة ليلى: lila_pro@work.net وهاتفها +00212 6611 87654321
3. المدير خالد: admin-office@company.org ورقم التواصل 00971 4433 99887766
"""

email_pattern = r"[\w.-]+@[\w.-]+\.[a-z]{2,4}"
phone_pattern = r"(?:00|\+)\d{1,3}.\d{4}.\d{8}"

emails_found = re.findall(email_pattern, data_source)
phones_found = re.findall(phone_pattern, data_source)

print("--- القائمة المستخرجة للإيميلات ---")
print(emails_found)
print("\n--- القائمة المستخرجة للهواتف ---")
print(phones_found)
```

### شرح العمليات البرمجية
* **استخدام re.findall:** على عكس search التي تأخذ أول نتيجة فقط، findall تمر على النص من بدايته لنهايته وتجمع كل ما يطابق النمط في قائمة واحدة.
* **مرونة نمط الهاتف:** الرمز `.` في النمط `\d{1,3}.` يعمل كفاصل مرن، أي أنه سيطابق الرقم سواء كان الفاصل بين أجزائه مسافة أو نقطة أو شرطة.
* **فائدة القوائم:** بمجرد تحويل البيانات المستخرجة إلى قائمة، يمكنك استخدام دوال بايثون عليها، مثل `len()` أو حلقات `for`.
